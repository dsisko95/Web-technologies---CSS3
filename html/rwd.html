Šta je RWD?

RWD (Responsive Web Design) odnosno prilagodljivi dizajn web strana
je vrsta web dizajna koja omogućava optimizovano prikazivanje, korišćenje, čitanje
i kretanje kroz veb sajt koji će biti pregledan na različitim uređajima.
Postoji nekoliko metoda kako bi se postigao RWD.

1. Viewport

Viewport je područje websajta koje korisnik vidi.
Problem kod ovog tipa RWD-a je bio taj što viewport ne može da "stane" na ekrane telefona i tableta.
Međutim, pojavio se novi meta tag koji rešava ovaj problem.
On izgleda ovako:
meta name="viewport" content="width=device-width, initial-scale=1"

Element meta zadaje pretraživaču instrukcije kako da kontroliše veličinu i skaliranje stranice.
Deo width=device-width podešava da širina stranice bude usklađena sa širinom uređaja.
Deo koda initial-scale=1.0 podešava inicijalni zum nivo kada stranicu učita pregledač.

2. Media upiti

Media upit je tehnika koja je predstavljena u CSS3.
Koristi @media pravila i u njima CSS kod i izvršava se ukoliko je određeni uslov tačan.
Većina web developera responsivni dizajn testira tako što sužava i širi browser.
Tada treba pratiti kako se elementi na stranici ponašaju i u skladu sa njima menjati media upite.

Primer (Pritisni F12)

Suzi širinu pregledača i kada prozor bude manji od 992 px, prikaži paragraf.

@media only screen and (max-width : 992px) { p#sakriven{ visibility: visible; } }

Ovaj paragraf će biti vidljiv ako je veličina prozora manja od 992 px.

Moguće varijacije sintakse media upita

@media (max-width:992px) - za prozor maksimalne veličine 992px primeni određeni stil.

@media screen and (max-width:992px) - za ekran uređaja i prozor max veličine 992px primeni određeni stil.
Sličan je iznad navedenom, sem razlike u ključnoj reči screen, koja može biti zamenjena sa print.

@media only screen and (max-width:992px) - reč "only" je namenjena da sakrije media upite za starije pregledače.

@media only screen and (?-width:?px) and (?-width:?px) - na mestu prvog upitnika moguće je zadati min ili max.
Na mestu drugog upitnika moguće je postaviti broj px tj. raspon npr. (min-width:800px) and (max-width:1200px)

Takođe, postoje mogućnosti za landscape i portrait opcije.

Primer (Pritisni F12)

Kada širina pregledača postane manja od visine, boja navigacionog menija će se promeniti u plavo.
Kod landscape opcije, širina pregledača je uvek veća od visine.

@media only screen and (orientation: portrait) { .navbar-default { background-color: #20B0D4; } }

3. Framework i grid view

Postoji mnogo CSS Framework-a koji nude RWD.
Besplatni su, laki za korišćenje i štede mnogo vremena.
Jedan od framework-a koji je korišćen za izradu ovog websajta je Bootstrap.

Šta je onda CSS framework?

CSS framework je gotov framework sa kojim je web dizajniranje uokvireno standardima korišćenjem CSS-a.
To znači da postoje gotovi HTML, CSS, JavaScript odnosno JQuery fajlovi koji trebaju da se izmene.
Većina framework-a koristi grid tj. mrežu.

Korišćenje mreže podrazumeva da je stranica podeljena na kolone, kojih obično ima 12.
Svaka kolona ima max-širinu od 100% i smanjiće se i proširiti ako se promeni veličina viewport-a.
